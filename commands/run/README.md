# RNA CLI

### Command: `run`

```sh
$ rna run --help

run  Trigger project script.
     <script>  The script to trigger.
```

This command is an alias to `yarn run` command.

### Usage
```sh
$ rna run test-saucelabs
$ rna run build-demo
```
