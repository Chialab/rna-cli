# RNA CLI

### Command: `remove`

```sh
$ rna remove --help

remove  Remove project dependencies.
        <module1> <module2> <module3>  The modules to remove
```

This command is an alias to `yarn remove` command.

### Usage
```sh
$ rna remove @dnajs/idom
```
